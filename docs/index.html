Sage

A blazingly fast, open-source terminal web novel reader. Scrape, cache, and read thousands of chapters offline — all from your terminal.

↓ Download v1.0.1 View on GitHub

Capabilities

Everything you need to read novels in the terminal

🛡️

Cloudflare Bypass Scraping

Uses primp with Chrome V144 JA3 fingerprint impersonation to seamlessly bypass Cloudflare protection and fetch entire chapter catalogs instantly.

📥

True Offline Reading

Download novels directly to a local SQLite database. Sage loads from disk first, letting you read anywhere without an internet connection.

📖

Premium Reader Experience

Customizable text width, margins, line spacing, paragraph style, color schemes (Sepia, Paper, Soft Dark), and text alignment — all adjustable in real time.

📚

EPUB Export

Export completed novels to well-formatted EPUB files for Kindle, Kobo, or any e-reader. Saved to your custom export directory.

💾

Storage Manager

An ncdu-style disk usage viewer. See per-novel storage, clear downloaded caches, and configure export paths — all without leaving the TUI.

🌐

Multiple Sources

Built-in support for NovelBuddy and NovelFire with an extensible provider trait. Search across sources and build a unified library.

Get Started

Install Sage

# Download the latest release for your platform
curl -LO https://github.com/musprodev/sage/releases/latest/download/sage-linux-x86_64.tar.gz
tar xzf sage-linux-x86_64.tar.gz
sudo mv sage /usr/local/bin/

# Verify installation
sage
# Ensure Rust toolchain is installed
# Note: On Linux, you may need sqlite-devel (or libsqlite3-dev) and openssl-devel (or libssl-dev)
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Clone and build
git clone https://github.com/musprodev/sage.git
cd sage
cargo build --release
sudo cp target/release/sage /usr/local/bin/
# Install build dependencies on Fedora
sudo dnf install sqlite-devel openssl-devel gcc pkg-config perl-FindBin

# Install Rust toolchain
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
source "$HOME/.cargo/env"

# Build from source
git clone https://github.com/musprodev/sage.git
cd sage
cargo build --release
sudo cp target/release/sage /usr/local/bin/
